## Color Scheme

### Stats Cards
- **Total Songs**: Gray/neutral tones
- **With Spotify**: Green (#16a34a)
- **Without Spotify**: Orange (#ea580c)
- **Coverage**: Dynamic based on percentage

### Song Status Badges
- **No Spotify**: Orange background (#ea580c/10), orange text, orange border
- **Synced**: Green background (#16a34a/10), green text, green border

### Interactive Elements
- **Primary Button** (Sync 50): Blue (#3b82f6)
- **Secondary Button** (Sync 20): Outline with blue border
- **Hover States**: Border color changes to primary/30

### Result Cards
- **Updated**: Green border (#16a34a)
- **Skipped**: Orange border (#ea580c)
- **Failed**: Red border (#dc2626)
- **Total**: Gray border

## Interaction Flow

1. **Initial Load**
   - Stats cards populate with current data
   - Pending tab shows songs without Spotify
   - Recently synced tab shows last 50 synced songs

2. **Sync Action**
   - Click "Sync 20" or "Sync 50"
   - Buttons disable, show spinning icon
   - Toast notification on completion

3. **View Results**
   - Result card appears below stats
   - Shows breakdown: updated, skipped, failed
   - Errors listed if any occurred

4. **Review Changes**
   - Switch to "Recently Synced" tab
   - See newly updated songs with album art
   - Click Spotify icon to verify data
