## üöÄ Quick Start

1. **Clone the repo**:

   ```bash
   git clone <repo-url>
   ```

2. **Install dependencies**:

   ```bash
   npm install
   ```

3. **Set up environment variables**:

   Copy `.env.example` to `.env.local` and fill in your Supabase credentials.

4. **Run the development server**:

   ```bash
   npm run dev
   ```
