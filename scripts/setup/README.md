# Setup Scripts

This directory contains scripts for setting up the Guitar CRM development environment and configuration.

## Available Scripts

### Environment Setup

- `setup-dev-env.sh` - Set up development environment
- `setup-dev-env-fixed.sh` - Alternative setup with fixes
- `setup-env.sh` - Configure environment variables

### Integration Setup

- `setup-github-actions.sh` - Configure GitHub Actions
- `setup-db.sh` - Initialize local database environment
