# Testing Scripts

This directory contains scripts for testing and quality assurance in the Guitar CRM project.

## Available Scripts

### E2E Testing

- `cypress-open.sh` - Open Cypress test runner in interactive mode
- `e2e.sh` - Run Cypress end-to-end tests headlessly

### Test Workflow

- `tdd-reminder.sh` - Display TDD guidelines and best practices
- `test-branch.sh` - Run tests for specific branches or categories
